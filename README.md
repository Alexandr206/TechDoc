# Система Навигации Автономного Робота

Данный документ описывает систему навигации для автономного робота, использующую алгоритм A*. Система позволяет роботу находить кратчайший путь от заданной начальной точки до конечной, учитывая препятствия на карте. Документ включает описание системы, руководство по установке и настройке, руководство пользователя, технические детали и примеры использования.
